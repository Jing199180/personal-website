URL

這個專案旨在為數碼暴龍機愛好者和收藏家提供一個直觀易用的在線平台，專門推廣數碼暴龍機維修服務。網站設計專注於清晰呈現修復流程，並包含快速預約系統，讓用戶能輕鬆提交維修需求。

Role:

  • Front-End Engineer
  • UX designer

Tech Stack:

  • HTML
  • CSS
  • Javascript
  • RWD

UX:

  • User Research
  • Information Architecture
  • Interaction Design
  • Visual Design
  • Figma

設計流程

需求與問題

需求：

  • 客戶目前有蝦皮的網站販售已修好的數碼暴龍機，但想要更進一步推廣這項維修服務。

問題：

  • 怪獸對打機的販售商品無保固服務，僅有售後服務，所以有問題的話只能找人維修。
  • 大部分的維修管道為臉書社團抑或是其他電商平台的賣家。
  • 不一定是販賣，有時候是互相交換的交易方式。

用戶研究

顧客：

  • 通過社交媒體和搜索引擎找到維修服務，經常在網上分享收藏品。
  • 對數碼暴龍系列有濃厚興趣，擁有多款數碼暴龍機。
  • 不熟悉維修渠道，擔心維修質量。

設計決策

方向：

  • 設計應簡潔明了，提供詳細的維修流程和服務介紹。增加用戶評價和案例展示，增強信任感。
  • “回憶”是這個網站的要點。

開發功能

主要功能

Javascript：

  • 導航滾動效果
  • 圖片輪播功能
  • CTA切換標籤及寄送表單

RWD：

  • 讓網站能夠適應各種不同大小的螢幕，提供最佳的瀏覽體驗。

主要功能展示

導航滾動效果

點擊導覽列中不同的按鈕，會有頁面翻轉到指定位置的效果。

圖片輪播功能

連續播放不同的案例照片，體現出維修的專業程度。

CTA切換標籤及寄送表單

方便的寄送聯絡資訊取得聯繫，並且連結到客戶的臉書社團，做進一步推廣。
